Fiche 02 — En production

Fight Data Arena

Un produit conçu, développé, déployé et maintenu seul — pas un projet d'école. En ligne sur fightdata.app.

Projet personnel · Full-stack · CI/CD

La frame data de Street Fighter 6, transformée en quiz

Les frame data sont d'ordinaire des tableaux bruts difficiles à mémoriser. Fight Data Arena les transforme en quiz interactifs : une API FastAPI sert 1 418 entrées en base PostgreSQL, alimentées par un scraper Python autonome, et un front Next.js les met en jeu.

  • Modèles Fighter / Move via SQLAlchemy, API REST FastAPI
  • Scraper BeautifulSoup alimentant la base depuis ultimateframedata.com
  • Punish Calculator : logique métier qui détermine si un coup est punissable selon son frame advantage au blocage
  • Déploiement continu : front sur Vercel, back sur Render, redéploiement à chaque push
Ouvrir fightdata.app GitHub

  • 2services déployés (front + API)
  • 100 %déploiement automatique au push
